Un archivo vivo de encuentros, preguntas y flyers

Esta página reúne los flyers del seminario como un catálogo navegable. Cada pieza sigue viviendo como su propio archivo HTML dentro de esta misma carpeta, y aquí aparece como una vista previa desplazable para recorrer las sesiones, comparar su lenguaje visual y entrar a cada charla.

Cómo navegar

Usa las fechas de arriba para saltar entre sesiones o desplázate horizontalmente por el catálogo. Cada flyer se carga desde su HTML original, así que esta página funciona como una portada-archivo, no como una copia.
